Web Design

APPLE FESTIVAL

As a project for INFO 1300, I created a mock website for the Ithaca Apple Harvest Festival. This project focused on improving visitor experience at the annual apple festival. I tested mockups of the website on multiple users in the demographic that I was targeting. My target demographic was out-of-town families who had minimal knowledge of Ithaca. Through user testing, I made multiple adjustments to the site until I had a finalized version that catered to the userâ€™s need. This site uses HTML5, PHP, and CSS.

A Family Friendly Tool

The finalized version of my website featured an array of bright pastel colors that emphasized feelings of fun and happiness. These colors were selected in order to create a family vibe. The fonts selected were intended to give the website a playful feel while still looking professional. The hope was to bring a simple tool for families interested in the event to use in order to both generate interest and as a navigational tool once there. The website is completely responsive for mobile devices. Thus, families will have no problem discovering any information they may need while at the festival. The images of the festival were taken by Andrew Halpern and other festival goers.
